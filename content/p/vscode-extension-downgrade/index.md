# VSCode 擴充功能降級方法

到擴充功能選單，點擊右下角的小齒輪

![Step 1](./step-1.JPG)

選擇 Install Another Version

![Step 2](./step-2.jpg)

選擇要的版本，大功告成！
> 這一步要等它讀取，會卡一下

![Step 3](./step-3.jpg)

畫面會多出一個 Update 按鈕，如果未來要升回去就按這顆。
